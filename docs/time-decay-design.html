🎯 设计目标

90天
保护期内高权重
180天
衰减至 5%
前慢后快
对数衰减曲线
  • 90天内：记忆权重保持较高，缓慢衰减（用户活跃期保护）
  • 90-180天：加速衰减，逐渐移除保护
  • 180天后：仅保留基础指数衰减，趋近于 0 但不完全消失

📊 新旧方案对比

旧方案

硬截断

  • 90天内：100% 权重
  • 90天后：0%（完全丢弃）
  • 第89天 = 第1天（无区分）
  • 第91天直接消失
新方案

对数衰减 + 保护常数

  • 0-30天：≈100%（几乎不衰减）
  • 30-90天：95% → 36%（缓慢下降）
  • 90-180天：36% → 5%（加速下降）
  • 180天后：5% → 0（渐进趋零）

📐 数学公式

最终得分公式（乘法组合）

Score(t) = clamp(BaseDecay(t) × (1 + BoostFactor(t)), minWeight, 1.0)

minWeight = 0.01（1% 保底，老记忆不会完全消失）

1. 基础指数衰减 BaseDecay(t)

BaseDecay(t) = e-t/τ

其中 τ = 60（半衰期60天）

2. 对数衰减增益因子 BoostFactor(t)

BoostFactor(t) = Bmax × max(0, 1 - ln(1 + t/k) / ln(1 + Tmax/k))

Bmax = 0.5（最大增益值）
k = 30（曲线形状参数，越小前期越平缓）
Tmax = 180（增益完全消失的时间）

为什么用对数？

对数函数 ln(1+x) 的特点是前慢后快：

  • x 较小时，ln(1+x) ≈ x，增长缓慢
  • x 较大时，增长加速
  • 这正好符合"早期保护、后期放开"的需求
