🧮 Embedding 模型升级 ✅ 已完成

❌ 原方案
all-MiniLM-L6-v2
  • 维度: 384
  • 中文支持: 差
  • 最大长度: 256 token
✅ 改进方案
BGE-M3 (BAAI/bge-m3)
  • 维度: 1024
  • 中文支持: 原生
  • 最大长度: 8192 token

🔍 Query Expansion 多路召回 ✅ 已完成

❌ 原方案
单查询检索 
chroma.query({
  queryTexts: [query], // 1个查询
  nResults: 20
})

召回率低，错过相关文档

✅ 改进方案
多路召回 + 同义词扩展 
// 原始: "budget 分配"
// 扩展: ["budget 分配",
//        "allocation quota",
//        "预算 配额"]

3-5 路并行查询，中英混搜

⏱️ 时间衰减优化 ✅ 已完成

❌ 原方案
90天硬截断 
// 90天前 = 完全丢失
if (age > 90) return null;

第1天和第89天权重相同

✅ 改进方案
对数保护 + 乘法衰减 
Score = Base × (1 + Boost)
// 90天: 40% 权重
// 180天: 12% 权重
// 最低: 1% 保底
